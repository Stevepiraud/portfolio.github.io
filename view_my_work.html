  • Orphée

    Orphée is a mobile application project which allows music producers to create music everywhere and whenever they want. The created projects can be transfer and converted to the music editing software they are using on their computer.

  • Ariane

    Ariane is a mobile application project which permits to geolocate French citizens and prevent them from risks of their hosting country by listing all necessary information and contacts if there are problems occuring in the country.

  • Wonderfood

    Wonderfood is a restaurant website project which allow the client to order his food online which will be delivered regarding the exact position they are. The colorful design of the mock-ups was done with the purpose of targeting students.
